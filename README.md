# ✋ Sign Language Recognition (ASL)

A deep learning-based web application that recognizes American Sign Language (ASL) alphabets from hand gesture images using a Convolutional Neural Network (CNN).

## 📌 Project Overview

This project is built as an end-to-end AI system:

- Multi-class CNN model for ASL alphabet classification
- Streamlit-based web interface for user interaction
- Cloud deployment using Hugging Face Spaces
- Version control and CI/CD using GitHub

---

## 🧠 Model Details

- Architecture: Convolutional Neural Network (CNN)
- Input size: 64 × 64 RGB images
- Classes: 29 (A–Z, del, nothing, space)
- Accuracy: **~88.7%**

---

## 📊 Dataset

- ASL Alphabet Dataset (Kaggle)
- Contains labeled images for each alphabet class
